# Authors Feature - Quick Start Guide

Get started with the Authors feature in 5 minutes!

---

## 1. Setup Groq (2 minutes)

### Get API Key

1. Go to: https://console.groq.com
2. Sign up (free)
3. Create API key
4. Copy the key

### Set Environment Variable

```bash
export GROQ_API_KEY=gsk_your_key_here
```

## What Happens Automatically

When you add a book:

```
1. Book added with author "John Smith"
2. System checks if author exists
3. If new: Author created
4. Author linked to book
5. Groq generates biography
6. Author appears on Authors page
```
