# Readloom v0.1.9

**Release Date**: November 8, 2025

## What's New

This version introduces the complete Authors feature with automatic synchronization and comprehensive author management.

## Key Features

- **Authors Feature** - Complete authors management system
- **Automatic Synchronization** - Authors sync when books are added
- **Author Profiles** - Beautiful card-based UI with author information
- **Author Search** - Search and add authors to your library

## Upgrading from v0.1.7

1. Update your installation
2. Restart Readloom
3. Database migration will run automatically
4. Authors will be synced from existing books

## Features

- Author profiles with photos and biographies
- Automatic author creation when importing books
- Author search across multiple sources
- Beautiful author cards and details modal
- Author statistics (books released, books owned)
