# Readloom v0.0.7

**Release Date**: Earlier version

## What's New

This version introduces the collection system and flexible organization capabilities.

## Key Features

- **Collection System** - Organize content into flexible collections
- **Collections Manager** - Create and manage multiple collections
- **Root Folders** - Link collections to multiple root folders
- **Setup Wizard** - Guide for first-time users
- **Improved E-book Scanning** - Better detection of files in existing folders
