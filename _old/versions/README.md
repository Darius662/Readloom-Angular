## What's in Each Version?

Each version folder contains:

- **FEATURES.md** - What's new in this version
- **Migration guides** - How to upgrade from previous versions
- **Specific documentation** - Features and fixes specific to that version
- **Known issues** - Issues specific to that version

## Finding Documentation for Your Version

1. Check the [CHANGELOG](../CHANGELOG.md) to find your version
2. Navigate to the corresponding version folder
3. Review the FEATURES.md and relevant documentation
