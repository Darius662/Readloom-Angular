# Metadata Providers

Readloom integrates with multiple metadata sources to provide accurate information about manga, comics, and books.

## Available Providers

- **[AniList](ANILIST_PROVIDER.md)** - Anime and manga metadata (recommended for manga)
- **[OpenLibrary](BOOK_PROVIDERS.md)** - Books and author information (recommended for books)
- **[Google Books](BOOK_PROVIDERS.md)** - Detailed book metadata
- **[MangaDex](METADATA_PROVIDERS.md)** - Manga database and search

## Getting Started

1. **[Overview](METADATA_PROVIDERS.md)** - Understand how providers work
2. **[Setup Guide](METADATA_PROVIDERS.md)** - Configure your preferred providers
3. **[Identifying Data Sources](IDENTIFY_SOURCE_QUICK_GUIDE.md)** - Know which provider was used

## Provider Comparison

| Provider | Best For | API Key Required | Notes |
|----------|----------|------------------|-------|
| AniList | Manga/Anime | No | Recommended for manga |
| OpenLibrary | Books | No | Recommended for books |
| Google Books | Books | Yes | Detailed metadata |
| MangaDex | Manga | No | Large manga database |

## Tips

- Use AniList for manga with accurate release dates
- Use OpenLibrary for books and author information
- Multiple providers can be enabled for redundancy
- No API keys required for AniList and OpenLibrary

For AI-powered author enrichment, see [AI Providers](../ai-providers/).
