# Troubleshooting

Help and solutions for common issues.

## Common Issues

- **[Known Issues](KNOWN_ISSUES.md)** - List of known problems and workarounds
- **[Authors Tab Issues](TROUBLESHOOT_AUTHORS_TAB.md)** - Debugging the Authors tab
- **[404 Errors](TROUBLESHOOTING_404_ERROR.md)** - Fixing page not found errors

## Debugging

1. **Check the logs** - Look in `data/logs/` for error messages
2. **Review [Known Issues](KNOWN_ISSUES.md)** - Your issue might already be documented
3. **Check [Technical Documentation](../technical/)** - For implementation details
4. **Enable debug logging** - See Settings for log level options

## Getting Help

1. Check relevant documentation in other sections
2. Review [Known Issues](KNOWN_ISSUES.md)
3. Check application logs
4. Open an issue on GitHub with:
   - Error message and logs
   - Steps to reproduce
   - Your Readloom version
   - Your system information

## Common Solutions

- **Restart the application** - Often fixes temporary issues
- **Clear browser cache** - For UI-related issues
- **Check API keys** - Ensure AI and metadata providers are configured
- **Verify file permissions** - For file system operations
- **Check database** - See [Database Schema](../technical/DATABASE.md)
