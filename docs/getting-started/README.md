## Choose Your Installation Method

### Docker (Recommended)
The easiest way to get started. See [Docker Setup](DOCKER.md).

```bash
docker compose up -d
```

### Manual Installation
For development or custom setups. See [Installation Guide](INSTALLATION.md).

```bash
pip install -r requirements.txt
python Readloom.py
```
